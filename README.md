## 📋 Features
- 🔐 Zero-knowledge AES-256-GCM encryption
- 💾 Offline-first IndexedDB storage with sync
- ⚡ Bun-powered blazing performance
- 📱 Installable PWA (add to home screen)
- 🎯 Drag & drop reordering + keyboard support
- 🎨 10+ themes (light/dark/OLED)
- 🏷️ Smart categories, filters, priorities
- 📅 Due dates & overdue alerts
- 🎓 Interactive onboarding tour

## 🛠 Self-Hosting (Advanced)
1. Clone repo: `git clone https://github.com/mohdadnanali/todo-app`
2. Install Bun: `curl -fsSL https://bun.sh/install | bash`
3. `cd todo-app && bun install`
4. `bun dev` for frontend, `bun run api` for backend
